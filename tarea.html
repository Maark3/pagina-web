¿Que es un Protocolos Http?

Es la base de cualquier intercambio de datos en la Web, y un protocolo de estructura cliente-servidor. protocolos HTTP nos permite realizar una petición de datos y recursos, como pueden ser documentos HTML.

¿Que es un Query String?

Éste es un termino que generalmente se utiliza para hacer referencias a una interacción con una base de datos. Es la parte de la URL que contiene los datos que deben pasar a las aplicaciones web.

¿Que es un Verbos HTTP?

En una arquitectura REST. Los verbos Http involucrados en un sistema REST son GET, POST, PUT, PATCH y DELETE.

¿Que es un DNS?

El sistema de nombres de dominio (DNS) es el directorio telefónico de Internet. Los navegadores web interactúan mediante direcciones de Protocolo de Internet (IP).El DNS traduce los nombres de dominio a direcciones IP para que los navegadores puedan cargar los recursos de Internet.

¿Que es un CDN?

Una red de distribución de contenido (CDN) es un grupo de servidores repartidos en distintas zonas geográficas que aceleran la entrega del contenido web al acercarlo a los usuarios.
